# /outreach

Draft a personalized cold outreach message in Dutch for a qualified lead, based on the cold-email framework.

## Usage

```
/outreach <company name or paste lead profile>
```

## Core Principles (from cold-email skill)

- Lead with **their situation**, not WAT? Agency's capabilities
- Sound like a peer-to-peer message, not marketing copy — the moment it sounds salesy, it's over
- Connect personalization directly to the pain point, not just as a compliment
- Single call-to-action only
- Eliminate every word that doesn't create curiosity, establish relevance, build credibility, or drive the ask

## Message Format for Tradespeople (owner/founder tier)

Tradespeople running small businesses (1–25 employees) are practical and time-poor. Target length: **5–8 sentences**.

Structure:
1. **Hook** — One specific observation about their business (Google rating, last Facebook post, missing website — something you actually found)
2. **Relevance** — Why this matters for them specifically (klanten vinden je niet, concurrentie staat hoger, etc.)
3. **Credibility** — One sentence about what WAT? Agency does (not a list, one sharp claim)
4. **Soft ask** — A low-commitment question or offer, not a pitch

## Subject Line Rules

- Short, reads like an internal message — not a newsletter
- No caps, no emojis, no "RE:" tricks
- Examples: `je Google-pagina`, `vraagje over je website`, `Mechelen renovatiebedrijven`

## Follow-Up Cadence (if no reply)

| Day | Angle |
|-----|-------|
| +4 | New evidence — show a competitor who improved their presence |
| +9 | Problem reframe — ask a diagnostic question |
| +16 | Social proof — brief customer result |
| +25 | Direct close — "Past dit niet, dan gewoon zeggen" |
| +35 | Final breakup — clean, no guilt |

Each follow-up is a new angle. Never "just following up."

## Output Format

Return:
1. **Subject:** `[subject line]`
2. **Body:** [the message in Dutch, plain text]
3. **Why this works:** [2–3 bullets explaining the personalization choices]
4. **Follow-up #1 draft** for Day +4

## Tone Reference

Write like a local Antwerp professional, not a corporate copywriter. Casual but sharp. Use "je/jij" not "u" unless the contact is clearly older or formal.
